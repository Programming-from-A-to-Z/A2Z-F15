Textarea element

This example shows how a user can enter text into a textarea element. Three events are demonstrated here:

  • An event is triggered whenever the user hits a key.
  • An event is triggered whenever the user hits enter.
  • An event is triggered whenever the user clicks the "submit" button.

In this example, the DOM elements are written into the HTML file and accessed from p5 using the select() function.
